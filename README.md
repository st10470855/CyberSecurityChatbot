# CyberSecurityChatbot

# Project Description
This is a simple Cybersecurity Chatbot built using C#.  
It helps users learn basic cybersecurity concepts like phishing, malware, passwords, and safe browsing through a console-based chat interface.



#  Features
- ASCII art interface
- Voice greeting on startup
- Interactive chatbot conversation
- Cybersecurity knowledge responses
- Typing animation effect
- Exit command system



# How to Run
1. Open project in Visual Studio  
2. Build solution  
3. Run the program
